RISU
E-Book Viewer

채팅을 책처럼, 몰입을 극대화하다

#RisuAI #E-Book #Plugin

[이미지 제안] PC와 모바일에서 이북 뷰어가 열려있는 목업 스크린샷
양면 펼침 레이아웃이 보이는 우아한 뷰어 UI 캡처

What is this?

RisuAI의 채팅 메시지를 실제 책을 읽는 듯한 경험으로 바꿔주는 뷰어 플러그인입니다.
긴 AI 응답도 페이지 단위로 나뉘어 편안하게 읽을 수 있습니다.

01 Core Features

📖

PC 양면 펼침

실제 책처럼 좌우 2페이지 레이아웃으로 몰입감 있는 독서 경험

📱

Mobile 최적화

스와이프 제스처로 페이지 넘김, 모바일에 최적화된 단일 페이지 뷰

🎨

다양한 테마

Light, Sepia, Dark 테마와 17종 한글 웹폰트 지원

🛠️

커스텀 CSS

자신만의 스타일을 CSS로 적용하여 완전한 커스터마이징

[이미지 갤러리 제안]
1. PC 양면 펼침 모드 스크린샷
2. 모바일 단일 페이지 뷰 스크린샷
3. 설정 메뉴 (테마, 폰트 선택) 스크린샷
4. 다크/라이트/세피아 테마 비교 이미지

03 Smart Navigation

Chat Index 이동

이전/다음 채팅 메시지로 빠르게 점프. 긴 대화도 쉽게 탐색

새 채팅 알림

AI 응답 수신 시 토스트 알림으로 새 메시지 안내

자동 이동

메시지 전송 시 자동으로 해당 채팅으로 이동

⚡ LB 모듈 완벽 지원

  • LB 모듈 버튼 목록을 뷰어 내에서 표시
  • 뷰어를 닫지 않고 바로 LB 모듈 실행
  • 리롤링 시 로딩 인디케이터로 상태 표시

04 How to Use

설치하기

1

RisuAI 설정 → 플러그인 관리 메뉴 진입

2

플러그인 파일 Import

3

채팅 입력창 옆 📖 책 아이콘 클릭!

🖥️ PC 조작

  • ← → 또는 PageUp/Down: 페이지 이동
  • Escape: 뷰어 닫기
  • 우측 하단 메뉴로 설정 변경

📱 Mobile 조작

  • 좌/우 스와이프: 페이지 이동
  • 하단 이전/다음 버튼
  • 상단 설정 버튼으로 설정 변경

[동영상/GIF 제안]
실제 사용 모습을 보여주는 짧은 데모 영상 또는 GIF
1. 뷰어 열기 → 페이지 넘기기 → 설정 변경 → 닫기 플로우
2. 모바일 스와이프 동작 GIF
